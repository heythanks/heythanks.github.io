JS里的数据类型转换

发表于 2019-02-14

JS中的七种数据类型分别是number,string,null,undefined,symbol,boolean,object。除了object，其他六种类型都是基本数据类型，object是复杂数据类型，由基本数据类型组成。

从其他类型转换为string

  • toString()方法
    使用toString()方法,number和boolean能正常转换为字符串,object运行出来结果为[object Object],null和undefined调用时报错，没有toString()属性。

  • String()构造函数

  • 老司机惯用 “” +

从其他类型转换为boolean

  • Boolean(‘1’)函数

  • 抖机灵方法
    使用取反运算，取反两次得到boolean值

对于Boolean值的总结除了五个falsy值为false,其他都为true。六个falsy值分别为0，’’,NaN,null,undefined，false。
